# Python

## Install Syntax Highlighting

```vim
:TSInstall python
```

## Supported language servers

```lua
python = { "jedi_language_server", "pylsp", "pyright" }
```

## Supported formatters

```lua
python = { "autopep8", "black", "isort", "reorder-python-imports", "yapf" }
```

## Supported linters

```lua
python = { "flake8", "pylint" }
```
